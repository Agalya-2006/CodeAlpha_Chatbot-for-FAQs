# Chatbot for FAQs

This project is a simple FAQ chatbot developed using Python.  
The chatbot answers user questions based on a dataset of frequently asked questions.

## Features
- Answers common questions automatically
- Uses Natural Language Processing techniques
- Matches user questions with the most relevant answer

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- TF-IDF Vectorization
- Cosine Similarity

## How It Works
1. The chatbot reads the FAQ dataset.
2. User questions are processed using NLP techniques.
3. TF-IDF vectorization converts text into numerical form.
4. Cosine similarity finds the closest matching question.
5. The chatbot returns the most relevant answer.

## Files in this Project
- chatbot.py – Main chatbot program
- faq.csv – Dataset containing questions and answers
- README.md – Project documentation
